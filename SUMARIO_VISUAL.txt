┌─ 💡 DICAS ──────────────────────────────────────────────────────────────┐
│                                                                           │
│  1. Nomes dos campos são CASE-SENSITIVE:                                │
│     "name" ≠ "Name" ≠ "NAME"  →  Use sempre minúsculo: "name"         │
│                                                                           │
│  2. Email DEVE ter @domain.com:                                         │
│     ❌ "usuario"  ❌ "usuario@"  ✅ "usuario@example.com"               │
│                                                                           │
│  3. Senha mínimo 6 caracteres:                                          │
│     ❌ "123"  ❌ "abcde"  ✅ "abcdef" ou "Senha123!"                    │
│                                                                           │
│  4. Role só aceita estes valores:                                       │
│     ✅ "admin"      ✅ "produtor"      ✅ "tecnico"                      │
│     ❌ "gerente"    ❌ "administrador" ❌ "técnico"                      │
│                                                                           │
│  5. Frontend deve exibir error.message:                                 │
│     ❌ alert('Erro ao cadastrar')                                       │
│     ✅ alert(`Erro: ${error.message}`)  ← Mostra mensagem específica    │
│                                                                           │
